## 무엇을 배우나

**노드 → 퀘스트 → 프로젝트**로 이어지는 프로젝트 중심 학습으로,
"복잡한 문법 대신 AI에게 말하듯 코드를 만드는" 바이브 코딩에서 시작해
직무별 맞춤형 AI 에이전트를 직접 기획하고 배포하는 경험까지 완성합니다.

### 커리큘럼 흐름

| 단계 | 기간 | 내용 |
|------|------|------|
| 기초 | 1–2주 | 온보딩 · 바이브 코딩 실습 |
| 기초 | 3–4주 | AI 모델 이해와 활용 |
| 기초 | 5주 | 바이브코딩 실전 프로젝트 (풀스택) |
| 심화 | 6주 | 프롬프트 엔지니어링 마스터 |
| 심화 | 7–8주 | 에이전트 기초 체력 다지기 |
| 심화 | 9–10주 | 에이전트 팀 꾸리기 (멀티 에이전트) |
| 심화 | 11–12주 | 에이전트 실전 프로젝트 |
| 프로덕션 | 13–15주 | AI 서비스 프로덕션 (배포·운영) |
| 프로덕션 | 16–24주 | 에이전트 프로덕트 파이널 프로젝트 (약 40일) |

### 기술 스택

- **웹**: Python, HTML/CSS, JavaScript, Svelte, Playwright, Vercel
- **AI**: LLM 활용, RAG, 프롬프트 엔지니어링, AI 에이전트 설계
- **도구**: Claude Code / Skill, OpenCode, 바이브 코딩
- **고급**: Rust, 커스텀 LLM 모델 배포
