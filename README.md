## 🧭 배우는 방식

강의를 듣기만 하지 않습니다. **노드 → 퀘스트 → 프로젝트**로 이어지는 프로젝트 중심 액티브러닝.

| 단계 | 이름 | 내용 |
|------|------|------|
| 01 | **노드(Node)** | LMS·교재로 개념을 스스로 깊이 파악하는 자기주도 학습 |
| 02 | **퀘스트(Quest)** | 조별 미션으로 배운 내용을 실제로 적용하고 아이디어 공유 |
| 03 | **프로젝트(Project)** | 기획→개발→UX→피드백→배포의 실무 프로세스로 결과물 완성 |

## 🗓️ 24주 커리큘럼

| 단계 | 기간 | 주제 | 핵심 키워드 |
|------|------|------|------------|
| 🟦 기초 | 1–2주 | 온보딩 · 바이브 코딩 | 개발환경, AI 페어 프로그래밍, Claude Code |
| 🟦 기초 | 3–4주 | AI 모델 이해와 활용 | LLM 개념, 토큰·컨텍스트, 모델 API |
| 🟦 기초 | 5주 | 바이브코딩 실전 프로젝트 | 풀스택 앱, 프론트/백 연동, 배포 |
| 🟪 심화 | 6주 | 프롬프트 엔지니어링 마스터 | 역할·맥락 설계, Few-shot/CoT, 출력 제어 |
| 🟪 심화 | 7–8주 | 에이전트 기초 체력 다지기 | 에이전트 루프, Tool Calling, RAG 기초 |
| 🟪 심화 | 9–10주 | 에이전트 팀 꾸리기 | 멀티 에이전트, 오케스트레이션, 워크플로 |
| 🟪 심화 | 11–12주 | 에이전트 실전 프로젝트 | 문제 정의, 설계, 도구 연동, 테스트 |
| 🟩 프로덕션 | 13–15주 | AI 서비스 프로덕션 | 배포, 모니터링, 비용·성능 최적화 |
| 🟩 프로덕션 | 16–24주 | **에이전트 프로덕트 파이널 프로젝트** | 40일간 기획→개발→배포, 시연용 포트폴리오 |
